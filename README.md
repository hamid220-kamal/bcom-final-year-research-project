# BCom Research Project: Impact of GST on Sole Trading Concern Business

This project is a comprehensive research study for the final year of B.Com (Computers) at **Vignan Degree College, Shadnagar**. It investigates the multidimensional impact of the Goods and Services Tax (GST) on sole trading concerns in the Shadnagar market region.

## 🎓 Project Details
- **Title**: Impact of GST on Sole Trading Concern Business
- **Institution**: Vignan Degree College (Affiliated to Palamuru University)
- **Place**: Shadnagar, Telangana
- **Academic Year**: 2025 - 2026
- **Supervised by**: K. Madhavi (Lecturer in Commerce)

## 📁 Project Structure

- `generate_pdf.py`: The main script that compiles all content, graphs, and survey data into the final 60+ page PDF report.
- `content.py`, `content2.py`, `content3.py`: Modular Python files containing the academic text, chapters, glossary, and statistical data.
- `survey_data.py`: Contains the detailed responses and metadata for the 15+ respondents surveyed in Shadnagar.
- `generate_graphs.py`: Script to generate analytical visualizations (Pie charts, Bar charts) based on survey results.
- `generate_questionnaire.py`: Helper script to generate the field survey instrument.
- `assets/`: Directory containing documentary photographs of traders and filled survey forms.

## 🚀 How to Run

### 1. Prerequisites
Ensure you have Python installed. You will also need the following libraries:
```bash
pip install fpdf2 matplotlib
```

### 2. Generate Graphs
To update the analytical charts based on the latest survey data:
```bash
python generate_graphs.py
```

### 3. Generate the PDF Report
To compile the final project report:
```bash
python generate_pdf.py
```
Numerical results and page counts will be printed to the terminal upon completion. The generated file will be named `BCom_GST_Project_Final.pdf`.

## 📖 Overview
This project serves as a complete digital ecosystem for a Final Year B.Com (Computers) thesis. It bridges the gap between **academic research** and **software automation** by dynamically generating a 60+ page professional PDF report from raw survey data.

---

## 📁 Project Structure

| Component | Description |
| :--- | :--- |
| **`generate_pdf.py`** | 🧠 **The Engine**: Orchestrates the entire PDF construction. |
| **`content.py` series** | 📝 **Knowledge Base**: Chapters 1-8 in modular Python variables. |
| **`survey_data.py`** | 📊 **Raw Data**: 15+ real-world respondent metadata and metrics. |
| **`generate_graphs.py`**| 📈 **Analysis**: Visualizes data into high-res PNG charts. |
| **`assets/`** | 🖼️ **Repository**: Field photos and survey documentation. |

## 🚀 How to Run

### 🛠️ 1. Setup Environment
```bash
# Clone the repository
git clone https://github.com/hamid220-kamal/bcom-final-year-research-project.git

# Install dependencies
pip install fpdf2 matplotlib
```

### 📊 2. Generate Visuals
Update the analytical charts based on real field data:
```bash
python generate_graphs.py
```

### 📄 3. Compile Final Manuscript
Compiles all data into a professional 60+ page B.Com thesis:
```bash
python generate_pdf.py
```
> **Output**: `BCom_GST_Project_Final.pdf`
